Recommended shadcn/ui Components

Components that work best for these designs

For FAQ Tab:

  • Accordion - Already using, perfect for collapsible Q&A (Options 1-4)
  • Card - Wrap each FAQ in its own Card component
  • Collapsible - Simpler alternative to Accordion for single items

For Info Tab:

  • Accordion - Best for collapsible sections with previews
  • Card + CardHeader - For consistent card structure
  • Collapsible - For custom expand/collapse behavior
  • Badge - For section type indicators (Essential, Updated, etc.)

My Recommendation:

Info Tab: Option 1 (Preview Text) or Option 4 (Left Border) - both show content preview

FAQ Tab: Option 1 (Basic Cards) or Option 4 (Floating) - clean individual cards
